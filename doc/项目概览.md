# ones-AI 平台 — 项目概览

## 项目简介

ones-AI Web 平台是一个**AI 智能工单处理系统**，允许用户通过 Web 界面批量提交 ONES 工单，由 AI（Claude CLI + ONES MCP）在远程服务器上自动分析和修改代码。

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | FastAPI + asyncpg + asyncssh + JWT |
| 前端 | Vue 3 + Vite + Element Plus + ECharts + xterm.js |
| 数据库 | PostgreSQL 16 |
| 部署 | Docker Compose（3 容器） |
| 通知 | 企业微信应用消息 |

## 核心功能

1. **ONES 登录**：邮箱 + 密码调用 ONES API 验证
2. **服务器管理**：SSH 凭证绑定与健康检查
3. **批量工单**：多工单同时提交，任务队列执行
4. **实时日志**：WebSocket 推送执行过程
5. **结果评价**：通过/不通过 + 原因记录
6. **管理仪表盘**：总览/趋势/用户/评价四个维度
7. **企微通知**：任务完成自动推送
8. **外部配置**：ONES/Gerrit/企微地址页面化管理
