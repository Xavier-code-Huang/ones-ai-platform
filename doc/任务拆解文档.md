## 任务总览

本文档将项目拆解为 **8 个里程碑**，共 **27 个任务**。每个任务标注了关联的需求 ID 和设计文档章节，确保实现不偏离规格。

**预计总工期**: 约 5-7 个迭代周期  
**依赖顺序**: M1 → M2 → M3/M4 并行 → M5 → M6 → M7 → M8

## M1 — 基础设施搭建

### T-001 项目初始化与脚手架

| 属性 | 值 |
|------|-----|
| **关联需求** | — (基础设施) |
| **关联设计** | §1.3 项目结构 |
| **优先级** | P0 |
| **依赖** | 无 |

**内容**:
1. 创建项目目录结构 `ones-ai-platform/backend/` + `ones-ai-platform/frontend/`
2. 后端: 初始化 FastAPI 项目，安装依赖 (`fastapi`, `uvicorn`, `asyncpg`, `asyncssh`, `python-jose`, `passlib`, `pydantic`)
3. 前端: 使用 `npm create vite@latest` 初始化 Vue 3 + Vite 项目（TypeScript 可选）
4. 安装前端依赖: `vue-router`, `pinia`, `element-plus`, `echarts`, `axios`
5. 配置前端暗色主题基础 CSS 变量
6. 后端创建 `config.py` 配置管理（环境变量读取）

**验收标准**:
- `cd backend && python main.py` 启动成功，访问 `/docs` 出现 Swagger 界面
- `cd frontend && npm run dev` 启动成功，浏览器显示首页
- 项目结构与设计文档 §1.3 一致
