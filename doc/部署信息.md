# Ones-AI 平台部署信息

## 服务地址
- **前端页面**: http://172.60.1.35:9621
- **后端 API**: 通过 nginx 代理 → 实际后端端口 **9625**
- **数据库**: PostgreSQL, 容器名 onesai-db, 端口 9627, 用户 onesai, 密码 onesai123, 库名 ones_ai_platform

## Docker 容器
| 容器名 | 作用 | 端口 |
|--------|------|------|
| onesai-frontend | nginx 静态文件 + 反向代理 | 9621 对外 |
| onesai-backend | FastAPI (uvicorn) | 9625 (host网络) |
| onesai-db | PostgreSQL | 9627→5432 |

> ⚠️ 后端 config.py 中 APP_PORT=9620，但 Docker CMD 覆盖为 `--port 9625`，以实际 9625 为准。
> nginx proxy_pass 指向 127.0.0.1:9625。
