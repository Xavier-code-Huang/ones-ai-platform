## 二、架构重设计

### 核心思路：三层架构 + 独立版本

```
┌──────────────────────────────────────────────────────┐
│                    共享基座层                          │
│  • Docker 镜像 (lango-claude:latest)                 │
│  • API Key 网关 (172.60.1.35:9601)                   │
│  • Claude CLI (内置于镜像)                            │
│  • 基础 Python 包 (requests/yaml/openpyxl)           │
└──────────────┬───────────────────────┬───────────────┘
               │                       │
    ┌──────────▼──────────┐  ┌────────▼──────────────┐
    │    lango-AI 产品层   │  │    ones-AI 产品层      │
    │                      │  │                        │
    │  Auth-Claude.sh      │  │  ones-AI.sh            │
    │  MCP 配置            │  │  ones_task_runner.py   │
    │  Hooks/Subagents     │  │  execute-task.md       │
    │  Engineer DNA        │  │  config.yaml           │
    │  host-run 代理       │  │                        │
    │  Lango Remote        │  │  ones-ai-platform/     │
    │                      │  │  (管理后台+前端)       │
    │  版本: v3.x          │  │  版本: v1.x            │
    │  用户: 全公司         │  │  用户: 平台自动化      │
    └──────────────────────┘  └────────────────────────┘
```

### 版本号体系

| 产品 | 版本格式 | 当前版本 | 变更节奏 |
|------|---------|---------|---------|
| **共享基座** (Docker 镜像) | `image-YYYYMMDD` | image-20260407 | 很少变 |
| **lango-AI** | `v3.x.y` | v3.8 | 每周 |
| **ones-AI** (CLI) | `v1.x.y` | v1.5.0 | 按需 |
| **ones-ai-platform** (Web) | `v2.x.y` | v2.1.0 | 按需 |

## 四、安装器改造

### install.sh 支持选择性安装

```bash
#!/usr/bin/env bash
# install.sh — 统一安装器
# 用法:
#   bash install.sh              # 安装全部 (lango-AI + ones-AI)
#   bash install.sh --lango      # 仅安装 lango-AI
#   bash install.sh --ones       # 仅安装 ones-AI
#   bash install.sh --image-only # 仅更新 Docker 镜像
```

这样你可以：
- 全公司服务器装全量
- 某台服务器只装 ones-AI（给平台用）
- 只更新镜像不动脚本

### 独立版本文件

```bash
# 安装完后
cat /opt/lango/lango-ai/version   # → 3.8
cat /opt/lango/ones-ai/version    # → 1.5.0
cat /opt/lango/image-version      # → image-20260407
```

一眼看清每个组件什么版本。


## 八、迁移风险评估

| 风险 | 影响 | 缓解措施 |
|------|------|---------|
| 目标服务器路径变化 | ones-AI 命令可能找不到 runner | install.sh 保持 `/opt/lango/ones_task_runner/` 路径不变 |
| task_executor 依赖 ones-AI 命令 | 平台执行可能中断 | `/usr/local/bin/ones-AI` 路径不变，无影响 |
| 旧安装器兼容性 | 旧版回滚可能出问题 | 保留 rollback 能力 |
| 两个 version 文件 | 安装验证需调整 | install.sh 分别打印两个版本号 |
